MaximumIntervalIntersectionProblem contains 3 projects:
	DemoMaximumIntervalIntersection is a simple console application that loads a text file "DemoData.txt" and outputs the answer.
	MaximumIntervalIntersectionProblem contains the solution code.
	TestMaximumIntervalIntersection is a project with NUnit tests for the solution code.
	
MaximumIntervalIntersectionUnity is a unity project that has a visualization of the solving process.
It was created in Unity version 5.4.0, so that will be required to open it.
